# チーム開発全体のスケジュール

スケジュールについては参考例であり、デプロイ以外のチェックリストのタスクを１周目（4/13）までに完了していれば構いません。

```
3/08: チームのリポジトリの作成、チームで自習室に集まりアイスブレイクと課題を行う日程を決める、個人でのGit学習
3/09: 個人でのGit学習
3/10: 自習室に集まり、課題1を着工（1回目）
3/11: 個人でのGit学習
3/12: 自習室に集まり、課題1を着工（2回目）
3/13: チームで作成する成果物を決定
3/14: チームメンバーが課題の内容を理解している状態、チェックリスト（リポジトリ）の提出
3/15: チーム開発着工
~~
3/23: 成果物提出
```
